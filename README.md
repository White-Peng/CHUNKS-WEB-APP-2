
# CHUNKS Web App 2

A React-based web application built from the CHUNKS Figma design. The original design is available at https://www.figma.com/design/TmHG1a73WgQY1ct67xCpsB/CHUNKS--Copy-.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
